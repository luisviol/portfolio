Overview

Batch Frame Creator is a Figma plugin that empowers designers to create multiple frames with customizable properties in one go, dramatically speeding up the design workflow.

Discovery & Problem Statement

While working on complex projects with extensive sitemaps, a significant pain point became apparent: Figma lacked a native function for batch creating frames. This was particularly evident when setting up initial layouts for large-scale projects. In that sense, based on my own experience with the platform and some comments from colleagues and friends led me to identify the need for this type of functions.

I identified and established some key pain points:
  • Manual creation of numerous frames with specific dimensions and names
  • Time spent on repetitive setup tasks instead of actual design work
  • Inconsistency in frame naming and sizing across large projects

Research

Taking into account the above and prior to the development of the plugin I did some research in the community to see if there was something similar. It was then that I started to download and test the functionalities of some plugins and define common functionalities or aspects that would allow me to "fill the gap".

  • Research some references, applying key words.
  • I looked for comments in forums that allowed me to have a better idea and consolidate concepts.
  • I verified limitations and scopes about what these offered.

Key findings:

  • Limited plugins addressing bulk frame creation
  • Lack of flexibility in naming and sizing options in existing solutions
  • High demand in community forums for this functionality
